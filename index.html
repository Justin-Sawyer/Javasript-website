The Premise

Ever had the feeling that you don't understand the article you're reading about JavaScript? Maybe the technical terms confuse you? Or that you're reading the code example, but cannot envisage in your head what is happening?

Maybe the explanation is simply "tl:dr"... Maybe you just want to see the code working in situ...

This site is designed to do just that.

It is a visual experiment, with almost no explanations or jargon to get bogged down in.

It is designed to give working examples, side by side, so that you can see the difference between a and b, x and y, substr() and substring()...

You are the host of the page - you decide the values you wish to play with.

While you're playing, you can choose to see the code working if desired.

As this site often shows comparisons and differences, the site is best viewed on a full size screen.
